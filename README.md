ObjectDex Privacy Policy

Welcome to the ObjectDex Privacy Policy. This document explains what data is collected by ObjectDex, how it is used, and who has access to it.

⸻

Glossary

* Application or Discord Application – Refers to the ObjectDex Discord bot, its associated owners, and the servers it operates in.
* Data – Refers to data stored by an individual bot instance, including personal data.
* Bot or Instance – Refers to a copy of the ObjectDex bot running independently with its own stored data.
* Collectible – Any virtual object intended to be collected through the bot, such as countryballs or future collectibles like virtual currency.
* Application Owners – The owners of the Discord application with access to the bot’s authentication methods and stored data. These users are commonly referred to as the ObjectDex Core Team or ObjectDex Team.
* ObjectDex Moderation Team or Moderation Team – The application owners and any additional users they appoint as moderators. A current list of moderators can be found on the official Discord server: https://discord.gg/RjzXFsS8kj

⸻

What Data Is Collected

ObjectDex collects the following information from Discord:

* User IDs – Used to identify users within the database.
* Server (Guild) IDs – Used to store server-specific settings required for the bot’s operation.
* Channel IDs – Used to determine which channels collectibles are allowed to spawn in.

Additionally, ObjectDex creates and stores the following information:

* The list of collectibles owned by each user.
* A history of collectible trades, including previous owners of each collectible.
* User and server settings, such as player preferences and the Spawn Silent option.

Message Content

The bot has access to message content solely for anti-cheat purposes.

* Message content is processed for less than one second.
* Message content is never permanently stored or cached.
* Application owners and moderators cannot read your messages through the bot.

⸻

How Data Is Stored

All persistent data is stored in a PostgreSQL database hosted on servers managed by BloxyBlitz.

⸻

Access to Data

Only Application Owners have access to ObjectDex’s stored data.

Application owners may access:

* The bot’s administrator interface.
* The database required to operate the service.

The ObjectDex Moderation Team does not have access to user data stored by the bot.

Stored data is never made publicly available and is protected using appropriate security measures.

⸻

Deleting Your Data

You may permanently delete your ObjectDex player data at any time by using the following command:

/player delete

This command removes your stored player data from the bot’s database.
